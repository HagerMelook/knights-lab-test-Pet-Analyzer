Website Features

📤 Easy Image Upload

Upload images via file input or drag-and-drop. Instantly previews your pet image for analysis.

🤖 AI-Powered Pet Classification

Automatically classifies whether your image contains a pet and identifies the breed using a trained ResNet model.

☐ Object Detection with YOLO

Uses YOLOv5 to detect pets and other objects in the image. Bounding boxes are drawn to visualize detected items.

🐈‍⬛ Semantic Segmentation

DeepLabV3 is used to generate a pixel-level segmentation mask for pets, giving precise object shape and region.

👥 Multi User Support

Support concurrent access to analysis multiple images by multi-user at the same time.

📄 Exportable Reports

Download a professional PDF report including classification results, detection boxes, and timestamped data.

🌙 Light / Dark Mode

Switch between light and dark themes dynamically to match your preference and improve accessibility.
